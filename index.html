HTML

  • The head element contains information about the webpage.
  • HTML attributes are always defined in the begining element, never the closing element.
  • The title, style, base, link, meta, and script metadata tags can be used inside the head element.
  • The body element represents the visible content shown to the user.
  • Elements nested inside the body element are known as child elements

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to stile a webpage using CSS: Inline CSS, Internal CSS Style Sheet, and External CSS Style Sheet.
  • Inline CSS is used when a developer wants to style an element directly in an HTML file. This is used to style a particular page element only.
  • Internal CSS Style Sheet, contains CSS rules for a webpage in the heading section of the HTML file. This is used to style multiple elements in th page code.
  • Exernal CSS Style Sheet, allows developers to keep all their CSS rules in a seperate file, which makes design changes easier.
  • Declarations contain two important components: The CSS property we want to apply and the value of the property.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch name: creates a new branch and switches to it.
  • git branch -a: lists all the branches availible on your project.
  • git pull branch name: Lets us recieve a branches modifications into the local environment.
  • git add -A: adds all modifications in the current working branch to the staging area.
  • git commit -m "______": lets us add a commit message that described what the commit contains.
  • git push origin branch name: lets us push the changes made locally to our remote github branch.

javaScrip

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A functions name is followed by parentheses ().
  • The consol.log method is used to make it easier for developers to test their code. The console.log is a test environment.
  • Console.log can be seen in action using Chrome DevTools, in the Console tab.
  • An array is a single variable that is used to hold a group of data.
  • Items are assigned an index according to the order they were stored in the array, due to indices following a predictable pattern of 0,1,2,3...
